Setting up

Download WindowsTerminal from github.

Color schemes

Generate a manifest of schemes

  1. Download the color schemes Color Schemes

  2. Use jq to form a large json file

hub clone mbadolato/iTerm2-Color-Schemes
cd iTerm2-Color-Schemes
jq -s '.' *.json > manifest.json

Default .json app

  1. Download Visual Code

  2. Create a dummy.json

  3. Select Visual Code as the default editor

Save default settings

  1. Open the default settings with Alt+Ctrl+'

  2. Copy all

  3. Open the non-default settting with Ctrl+'

  4. Paste the default one

Add color schemes

  1. Open the manifest.json from above

  2. Splice into the scheme section

  3. Make sure to delete the inbetween square brackets

Reformat the .json file

  1. In Visual Code, use Shift+Alt+F to reformat the document

  2. This can also be done by right clicking and getting the "context menu"

Change the color scheme

  1. Under a profile add a "colorScheme": "Broadcast" (or whatever)

  2. Using Notepad++ and content aware filling

  3. C:\Users\adam\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json

  4. Open the above file in Notepad++

  5. When selecting a colorScheme, Notepad++ seems to know the available choices

  6. I am primarily using Visual Code for the formatting capability, other formatters won’t work on the settings.json file because they contain comments, but Visual Code doesn’t seem to mind the comments.

Adding a font

Adding a font (e.g. Ubuntu Monospace)

  1. Installing the font is a pain.

  2. Download a mono spaced font from fontsquirrel.

  3. Unpack it and right click on a ttf file to install it.

  4. Have to use regedit to add the font under.

    1. Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Console\TrueTypeFont

    2. Insert a new String Value with a unique ID, and use the font name 'Ubuntu Mono'.

      1. The font name can be found when you right click on the .ttf and try to preview the font.

Problems

  1. Fonts don’t render well.

  2. Color scheme change not working (it is not clear how to change the palette).

vs WindowsTerminal

  1. The default font looks great.

  2. After figuring out how to add the color schemes, eye strain is minimal.

  3. The response is quite smooth.
